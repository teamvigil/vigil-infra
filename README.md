# Vigil Infrastructure

Ce dépôt contient toute la définition de l'infrastructure Vigil sous forme de code (Terraform, Ansible).

## Pré-requis
- Terraform >= 1.5
- Un compte AWS/Azure/GCP configuré

## Déploiement
1.  `cd environments/staging`
2.  `terraform init`
3.  `terraform plan`
4.  `terraform apply`
